# Privacy Policy — speedctl

**Last updated:** April 1, 2026

## Overview

speedctl is a Chrome extension that controls video playback speed on YouTube. It does not collect, transmit, or share any user data.

## Data Collection

This extension does **not** collect any of the following:

- Personally identifiable information
- Browsing history or web activity
- Health, financial, or authentication information
- Location data
- Personal communications
- Website content

## Data Storage

The extension stores only user-configured settings (default speed, channel rules, regex rules, and keyboard shortcuts) using Chrome's built-in `storage` API. This data remains entirely on the user's device (or synced via their Chrome account if they choose sync storage) and is never sent to any external server.

## Third-Party Services

This extension does not communicate with any external servers or third-party services.

## Permissions

- **storage**: Used solely to save and retrieve extension settings.
- **Host permission (youtube.com)**: Used solely to run a content script that adjusts video playback speed on YouTube pages.
